##  Assignment Description

This assignment focuses on implementing **Stack** and **Queue** data structures using C.
It covers basic operations and real-world applications of both concepts.

---

##  Topics Covered

###  Stack (LIFO - Last In First Out)

* Stack follows **Last In First Out** principle
* Operations used: **Push** and **Pop**

**Programs:**

* Reverse a String using Stack
* Balanced Parentheses Checking
* Next Greater Element

---

###  Queue (FIFO - First In First Out)

* Queue follows **First In First Out** principle
* Operations used: **Enqueue**, **Dequeue**, **Display**

**Programs:**

* Queue using Linked List
* Printer Queue Simulation

---

##  Key Concepts

* Efficient use of Stack for string reversal and expression checking
* Use of Stack to solve Next Greater Element problem
* Implementation of Queue using dynamic memory (Linked List)
* Understanding real-life applications like printer job scheduling

##  Conclusion

This assignment helped in understanding how Stack and Queue work internally and how they are used to solve practical problems efficiently.
